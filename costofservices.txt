Fly.io:

	Free tier gives you:

	3 shared-1x CPU VMs (~2340 hours/month in total)

	3 GB persistent storage

	If you run 1 small VM 24/7, it fits well inside free tier — no charges.
	
	Uses 1 shared CPU → free

	Uses 256MB RAM → free

	1 region → free

	Low bandwidth → free

	You’ll only be charged if you:

	Run more than 3 VMs simultaneously, or

	Exceed persistent volume limits, or

	Use any paid features outside the free tier.
