Testing:

-in testing two type of testing is there one is manual testing , automation testing (selenium)
-ManualTesting:
	Two type is there functional and non functional
-In functional contain:
	-retest - If tester saw any bugs then he inform to the development team then again test that
	-regression - If tester saw any issues in the module that is called regression
	-sanity - If the tester saw any problem in the page of the module that is called sanity test.
	-smoke testing - basic level testing like install and login
	-exploratory testing - blind user level testing which means test without any knowledge of the app
	-unit testing - Module or component based testing
	-Integration testing - to test one page changes reflect on another one.
	-system testing - Overall one workflow of app
